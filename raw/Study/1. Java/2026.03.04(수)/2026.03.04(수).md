- 단순 if (If01 클래스)
```java
if(조건식){
	if 조건식에 성립될때 실행될 구문
}
```

(실습)
```java
int su = 4;

if (su % 2 == 0) {
	System.out.println(su + "는 짝수입니다."); // 조건식에 성립되면 실행되는 구문 (단순 if구문 안에 있으니까)
}

System.out.println("나는 무조건 실행 됩니다."); // 조건식의 성립유무와 상관없이 실행되는 구문 (단순 if구문 밖에 있으니까)
```

(출력)
4는 짝수입니다.

- 양자 택일 (if~else) (경우의 수 2개) (If02 클래스)
```java
if(조건식){
	if 조건식에 성립될때 실행될 구문
}else{
	if 조건식에 성립되지 않을때 실행되는 구문
}
```

(실습)
```java
int su = 3;

if(su%2==0){//짝수이면
	System.out.println("하하하");
}else{//홀수이면
	System.out.println("호호호");
}
```

(출력)
호호호


- 다중 택일 (if~else if~else)(경우의 수 2개 이상) (If03 클래스)
else if라는 메소드가 있는게 아니고
그냥 else라는 메소드 뒤에 if라는 메소드가 또 사용된 것

(실습)

# 3. 11:30 ~ 12:20
- switch (switch 메소드) (일반적으로 switch case 구문이라고 불림)
```java
switch(특정값){
	case 특정값:
		특정값이 성립될때 실행되는 case 구문
}
```


- if구문이랑 다른점
if구문은 다양한 조건식 중 하나의 조건식에 해당한 구문만 수행하지만
switch 구문은 조건에 맞는 case 구문을 만나면 그 구문부터 밑에 있는 나머지 case 구문들을 쭉 수행함

switch 구문은 조건에 맞는 case 구문을 만나 수행하고 그만두려면 break 메소드를 추가해야 함.


(실습)
```java
int su = 5;

switch (su){
	case 1:
		System.out.println("홀수");
		break;
	case 2:
		System.out.println("짝수");
		break;
	case 3:
		System.out.println("홀수");
		break;
	case 4:
		System.out.println("짝수");
		break;
	case 5:
		System.out.println("홀수");
		break;
	case 6:
		System.out.println("짝수");
		break;
}
```

(출력)
홀수

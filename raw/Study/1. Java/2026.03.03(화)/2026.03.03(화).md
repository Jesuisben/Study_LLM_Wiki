- 조건 연산자 (3항연산자) (P.28)
// 2개의 숫자 중에서 큰수 고르기
int x = 10, y = 3;

(조건식) (조건연산자)(3항연산자)
타입 result = 조건식 ? 참일때처리 : 거짓일때처리 ;
사용 형식 : result = Condition ? truepart : falsepart ;
변수(result)에 대입(할당)되는 것은 결국 truepart 혹은 falsepart니까
\*변수와 truepart, falsepart가 같은 타입의 데이터여야 한다!!\*

(이렇게도 이용 가능)
원래 조건식은 참 / 거짓 같은 2가지 경우의 수만 표현 되는데
참이나 거짓 안에 또 다른 조건식을 넣으면 무제한의 경우의 수 표현 가능
ex)
사용 형식 result = x>y ? x : (x == y ? 0 : y) ;

(연필로 적어보기)
```
int x = 10, y = 20;

// 2개의 숫자 중에서 큰수 고르기
// 타입 result = 조건식 ? 참일때처리 : 거짓일때처리 ;
int result = x >= y ? x : y ;
System.out.println("result : " + result);
```

- 형변환 (Casting)
- 암시적 형변환
타입 변경 (ex 정수 -> 실수)
ex) double d = 100;
출력 -> 100.0
정수 100을 정의 및 대입하면 실수 데이터로 나옴 (정수데이터가 실수데이터로 형변환이 이루어짐) (Java 가상 머신에 의해서)
(실수가 정수를 포함한 더 큰 집합이니까 가능)

(암시적 형변환 불가능 사례)
int i = 3.14;
System.out.println("i : " + i);
-> 에러(incompatible type)
실수 데이터가 정수 데이터로 형변환이 이루어지는 일은 없음
(정수가 실수에 포함된 더 작은 집합이니까 불가능)

- 명시적 형변환
형변환연산자(캐스트 연산자, 캐스팅 연산자)(ex - (int))에 의한 형변환
```java
int i = (int)3.14;
System.out.println("i : " + i);
```
출력 -> i : 3

- Java에서의 나누기
Java에서 정수 나누기 정수는 정수가 나온다.

- **!!(명시적) 형변환 이용!!**
1.
System.out.println(14/5); -> 출력 : 2
System.out.println((double)14/5); -> 출력 : 2.8
System.out.println((double)14/5); -> 14/5 -> 14.0/5(명시적 형변환)
-> 14.0/5.0(정수의 실수화 - 암시적 형변환) -> 2.8

2.
System.out.println(14/5); -> 출력 : 2
System.out.println((double)(14/5)); -> 출력 : 2.0
System.out.println((double)(14/5)); -> 14/5 -> 2 -> 2.0(명시적 형변환) -> 2.0

- 형변환 응용
int kor = 50, eng = 60, math =80;
int total = kor + eng + math;
System.out.println("총점 : " + total);

1번 경우
double average = total / 3;
System.out.println("평균 : " + average);
-> total/3이 (산술 연산자) 먼저 진행되어서 63 -> 그 후 대입 연산자가 진행되어서 double에 의해 암시적 형변환이 이루어짐

2번 경우
double average = (double)total / 3;
System.out.println("평균 : " + average);
-> (double)total로 인해 190이 190.0으로 명시적 형변환이 이루어짐 -> 분자가 190.0으로 실수이기에 분모도 실수로 바뀌는 암시적 형변환이 이루어짐 -> 이로인해 출력
-> 평균 : 63.333333333333336

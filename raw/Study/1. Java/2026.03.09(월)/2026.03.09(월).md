# (클래스 안의) 메소드 예시 (PDF p.61)
```java
// '매개변수'는 메소드 외부에서 넣어 주는 값입니다. (매개 변수 (투입변수))
문법 :
반환타입 메소드이름(매개변수리스트){
	내가 하고자 하는 일 ;
	return 반환할값 ;
}
-> [반환타입]을 가진 [메소드]가 [매개변수리스트]가 투입될때 [내가 하고자 하는 일]을 z실행한다
```

```java
ex) int plus5(int x){,,,} / int plus5(int x, int y){...} - 매개변수리스트라고 한 이유가 매개변수가 2개 이상일 수도 있어서 ( ,(콤마)로 연결 )
ex)
int plus5(int x){
	int result = x + 5;
	return result; //투입된 매개변수리스트로 인해 실행되어 나온 값을 반환타입을 가진 값으로 반환한다.
}
```

1) 메소드 이름 정하기 1)plus5
2) 매개 변수의 갯수 및 타입 정하기 2)int x
3) 반환 타입 (결과 타입) 정하기 3)int (plus5 옆에있는)

*변수는 소괄호가 없고 / 메소드는 소괄호가 있음

shin. (f)price     			 int
		(m)plus5 (int x)     int

```java

```


```java
int result = shin.plus5(su); //클래스에서 반환된 값을 반환받으려고 새로운 변수에 할당
```

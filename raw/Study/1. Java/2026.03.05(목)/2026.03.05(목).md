# 1. 09:30 ~ 10:20
- 반복문 for
* 식들 사이에 세미콜론 주의

```java
for(초기식 ; 조건식 ; 증감식){
	//반복 수행될 문장
}
```

제어변수 : i

초기식 : 최초 수행되는 수식 (~부터)(오직! 한번만 실행됨) - int i = 1 (1부터)
조건식 : 비교/판단을 위한 근거가 되는 수식 (조건식이 성립(참)되면 실행) - i < 11 (10까지)
증감식 : 제어 변수 증가/감소를 위한 수식 (제어변수를 ~정도 증감해라) - i = i + 1 / i += 1 / i++ (1씩 증가해라)

* 초기식부터 조건식이 성립(참)될때까지 반복해서 실행됨

```java
//풀어서 보기
total(생성) i(생성)
0		1
0+1		2
1+2		3
3+3		4
6+4		5
10+5		6
15+6		7
21+7		8
28+8		9
36+9		10
45+10	11 (i가 소멸된다.)(생성X)(조건식 거짓)
출력
total(소멸)
```

# 3. 11:30 ~ 12:20
- 반복문 while

```java
초기식 ;
while(조건식){
	반복적으로 실행할 코드
	증감식 ;
}
```

(실습)
```java
// 1부터 10까지 숫자의 총합
int total = 0;
```

```java
int i = 1;
```

```java
while (i<11){
	total += i;
	i++;
}
```

```java
System.out.println("총합01 : " + total);
```

(출력)
총합01 : 55

(실습)
```java
// 1부터 100까지 3씩 증가하는 수의 총합
// 1 + 4 + 7 + ... + 100 = 1717
int total = 0 ;
int i = 1; //초기식
while (i<=100){//조건식
	total += i;
	i += 3;//증감식
}
```

```java
System.out.println("총합02 : " + total);
```

```java
// 97부터 2까지 5씩 감소하는 수의 총합
// 97 + 92 + 87 + ... + 2 =990
total = 0;
i = 97;
```

```java
while (i>=2){
	total += i;
	i -= 5;
}
```

```java
System.out.println("총합03 : " + total);
```

```java
// 1부터 100까지 5씩 증가하는 수의 제곱의 총합
// 1*1 + 6*6 + 11*11 + ... + 96*96 = 63670
total = 0;
i = 1;
```

```java
while(i<=100){
	total += i*i;
	i += 5;
}
```

```java
System.out.println("총합04 : " + total);
```

(출력)
총합02 : 1717
총합03 : 990

### # class 선택자 (전체에서 몇개만 그룹핑할때 사용)
<style>
	ol{list-style-type: upper-alpha;}
</style>
-> 원래 이건데 -> 이렇게 하면 ol 전체가 다 list-style-type: upper-alpha;가 됨

<ol class="upper-alpha">가 적힌 부분만 바꾸기 위해서는 * 마지막에 ; (세미콜론) 적으면 안됨!! (중요!!) *
class 선택자는 ol라고 태그명을 적기보다는 .(dot)+속성값으로 대체하면 됨 ( .class의_속성값 )
-> 주로 2개 이상의 항목에 대하여 동일한 스타일을 지정할 때 사용

따라서
->
<style>
	.upper-alpha{list-style-type: upper-alpha;}
</style>
이렇게 하면 <ol class="upper-alpha;"> 부분만 list-style-type: upper-alpha;로 바뀜

* 2개 이상의 속성을 가질 수 있음 (구분자 : 스페이스) (중요!!!)

<태그명 class="속성 속성"> ..... </태그명>
ex)
<li class="myyellow myblue">사과</li>

(<head>)
<style>
	.myyellow{color: yellow;background-color: black;}

### # top-down 방식
위에서 아래로 순차 진행

<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        /* 태그 선택자 */
        li {                                  // li태그를 전체 적용
            font-size: 12px;
            color: purple;
        }
        /* class 선택자는 dot 기호를 사용합니다. */
        /* 주로 2개 이상의 항목에 대하여 동일한 스타일을 지정할 때 사용 */
        .upper-alpha{list-style-type: upper-alpha;}                         // upper-alpha인 class만 적용
        .myyellow{color: yellow;background-color: black;}            // myyellow인 class만 적용
    </style>
</head>

-> 처음에 색이 purple로 바뀌었다가 나중에 밑에 있는 코드에 color가 yellow여서 yellow로 바뀜

* 헷갈리지 말기 *
-  태그 속성 이용                  VS          태그 선택자 이용
<li style="color: purple;">    VS      <style>    li{color: purple;}    </style>

- 태그 선택자 + class 선택자
<li class="myyellow">       ------>    <style>    .myyellow{color: purple;}    </style>

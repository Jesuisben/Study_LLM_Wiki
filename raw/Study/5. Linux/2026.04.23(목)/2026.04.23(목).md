#### \# VI Editor (Linux 이론.pdf(P.99))
- VI Editor의 3가지 모드

- 명령 모드(기본 모드)
커서 이동, 수정, 삭제


- 입력 모드
명령 모드에서 (a, i, o)글자 중 1개를 입력하면 입력모드로 전환됨
a (Append) : 커서 뒤에서부터 입력
i (Insert) : 커서 앞에서부터 입력
o (Open line) : 커서 아래에 새 줄을 만들며 입력

esc누르면 다시 명령모드로 전환

1) 입력한 것을 저장하려면
-> esc누르고(명령 모드로 가서) :wq (Write Quit - 저장하고 나가기)를 입력하면 됨

2) 저장하지 말고 빠져 나가려면
-> esc누르고(명령 모드로 가서) :q! (Quit - 나가기)를 입력하면 됨


- 실행 모드
검색, 종료, 명령어(저장, 읽기, 외부 명령 실행 등)
명령 모드에서 (:, /, ?)글자 중 1개를 입력하면 실행모드로 전환됨

esc누르면 다시 명령모드로 전환

### # 그룹(집계) 함수 (Oracle 교안 P.296) (대략 5개 정도 있음)

- avg : 평균, 널 값은 무시 / 숫자 가능 / 문자 및 날짜 불가능
ex) stock : 10, 20, 30, null -> 60나누기 3 -> null값을 무시함.
-- 평균 가격

- count

2) count(expr) : null 값을 제외한 개수를 표현합니다.  -> stock -> 3개
3) count(distinct expr) : 중복된 데이터는 하나로 개수를 표현합니다. -> stock -> 3개

응용 : null값인 price컬럼의 데이터 갯수
SELECT count(*) - COUNT(PRICE) AS "단가널갯수" FROM PRODUCTS; -> 쌍따옴표는 파생컬럼의 이름을 정할때 사용함

SELECT MAX(PRICE) AS MAX_PRICE FROM PRODUCTS;
응용 : 문자열을 숫자로 생각해서 가나다 순 이름 찾기
SELECT max(name) FROM members ;

- min : 최소 값, 널 값은 무시 / 숫자 가능 / 문자 및 날짜 가능
SELECT MIN(PRICE) AS MIN_PRICE FROM PRODUCTS;
응용 : 문자열을 숫자로 생각해서 가나다 순 이름 찾기
SELECT min(name) FROM members ;

- sum : 합계, 널 값은 무시 / 숫자 가능 / 문자 및 날짜 불가능
-- 전체 재고 합계

### # group by 함수 (Oracle 교안 P.309)
특정 컬럼 기준 데이터 그룹화
그룹화할컬럼은 범주형 데이터여야 함 (연속형 데이터는 안됨)

문법 :
select 일반컬럼명, 그룹함수(컬럼명)
from 테이블명
group by 그룹화할컬럼명
ex)
- 남녀 각각 몇명인가요?
SELECT gender, count(*)
FROM members
GROUP BY gender ;

- 카테고리별 상품 개수
SELECT category, count(*)
FROM products
GROUP BY category ;
-> 사용법 : 그래프, 차트같은 시각적 자료 표현할때 유용함

- 카테고리별 평균 가격
SELECT CATEGORY, AVG(PRICE) AS AVG_PRICE
FROM PRODUCTS
GROUP BY CATEGORY;

- 카테고리별 최고 가격 (높은 순)
SELECT CATEGORY, MAX(PRICE) AS MAX_PRICE
FROM PRODUCTS
GROUP BY CATEGORY
ORDER BY MAX_PRICE DESC;

-- 카테고리별 회사별 최고 가격(높은 순) / 2개 이상의 그룹화 + 정렬
SELECT category, company, max(price) AS max_price
FROM products
GROUP BY category, company
ORDER BY category, company desc;

### # group by 함수 + having 절
having은 group by 함수에다가 필터링(where함수같은)을 하는 것, 조건을 더 추가하는 것
having절 안에는 단일행 함수는 못쓰고 그룹 함수만 쓸 수 있음 <-> where 절에서는 그룹 함수를 못쓰고 단일행 함수를 쓸 수 있음
-> having절에는 avg / count / max / min / sum
having은 group by에 종속되어 있음
문법 :
group by 그룹화할컬럼명 having 조건식

ex) 회사별 상품 개수를 파악하되 3개이상인 회사 정보만 출력
SELECT COMPANY, COUNT(*) AS CNT
FROM PRODUCTS
GROUP BY COMPANY
HAVING count(*) >= 3
ORDER BY company;

### # 조인(Join) (Oracle 교안 P.348)
서로 다른 두 개 이상의 테이블을 **공통된 컬럼(연결 고리)**을 기준으로 합쳐서,

방향성 없음 - 양방향

- 아우터 조인 (Outer Join) (서로 다른 두 개 이상의 테이블들의 차집합)

작성시 처음 적는 것이 기준이 됨
기준에 따라 2개로 나뉨
1) 래프트 아우터 조인

2) 라이트 아우터 조인
오른쪽 테이블 + 테이블들의 교집합

3) 풀 아우터 조인 (서로 다른 두 개 이상의 테이블들의 합집합)
왼쪽, 오른쪽 테이블의 합집합

- 다차원 조인 : 3개 이상 조인 하는 것

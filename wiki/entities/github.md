# GitHub

## 무엇인가

GitHub는 [[entities/git|Git]] 저장소를 온라인에 보관하고 공유할 수 있게 해 주는 웹 기반 개발 협업 서비스다.

## 이 위키에서의 맥락

Java 수업 초반에 `MyJava` 프로젝트를 원격 저장소에 올리기 위해 처음 등장했다. 로컬 프로젝트를 Git으로 초기화한 뒤 GitHub 저장소 URL을 IntelliJ IDEA 또는 Git 명령과 연결하는 흐름으로 학습했다.

## 핵심 기능 / 특징

- 원격 저장소 제공: 내 컴퓨터 밖에 Git 저장소를 보관한다.
- 백업: 컴퓨터나 프로젝트 폴더에 문제가 생겨도 원격 저장소에 코드 이력을 남길 수 있다.
- 공유와 협업: 다른 사람과 같은 저장소를 보고 작업할 수 있다.
- IDE 연동: IntelliJ IDEA 같은 개발 도구에서 GitHub 저장소를 연결해 변경사항을 관리할 수 있다.

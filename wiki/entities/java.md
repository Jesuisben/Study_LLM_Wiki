# Java

## 무엇인가

Java는 객체지향 프로그래밍 언어이며, 이 위키에서는 국비지원 웹서비스 풀스택 과정의 첫 번째 주요 학습 언어다.

## 이 위키에서의 맥락

교육 과정 초반에는 IntelliJ IDEA 설치와 Java 프로젝트 준비를 먼저 진행했고, 이후 변수, 자료형, 연산자 같은 기초 문법으로 들어갔다. Java 학습은 이후 Spring, Spring Boot 백엔드 학습의 기반이 된다.

## 핵심 기능 / 특징

- 정적 타입 언어: 변수를 선언할 때 타입을 정한다.
- 객체지향 언어: 이후 클래스, 객체, 메서드, 상속 같은 개념으로 확장된다.
- 백엔드 생태계와 연결: Spring/Spring Boot 학습의 기반 언어다.
- IDE 활용: 수업에서는 IntelliJ IDEA를 Java 개발 도구로 준비했다.

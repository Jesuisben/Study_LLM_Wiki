# DBeaver

## 무엇인가

여러 데이터베이스에 접속해 SQL을 실행하고 스키마를 관리할 수 있는 데이터베이스 IDE다.

## 이 위키에서의 맥락

국비지원 웹서비스 풀스택 과정의 수업 노트에서 반복적으로 등장하는 기술/도구로, 날짜별 요약과 개념 페이지를 연결하는 기준점이다.

## 핵심 기능 / 특징

- Oracle 실습 도구
- SQL 실행과 결과 확인
- ERD/관계도 확인
- 사용자와 테이블 작업 보조

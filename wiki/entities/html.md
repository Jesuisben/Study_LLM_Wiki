# HTML

## 무엇인가

웹 문서의 구조를 표현하는 마크업 언어다.

## 이 위키에서의 맥락

국비지원 웹서비스 풀스택 과정의 수업 노트에서 반복적으로 등장하는 기술/도구로, 날짜별 요약과 개념 페이지를 연결하는 기준점이다.

## 핵심 기능 / 특징

- 태그와 속성
- 목록, 표, 이미지, 폼
- 브라우저가 해석하는 문서 구조

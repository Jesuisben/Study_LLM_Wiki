# jQuery

## 무엇인가

JavaScript DOM 선택과 이벤트 처리를 간결하게 도와주는 라이브러리다.

## 이 위키에서의 맥락

국비지원 웹서비스 풀스택 과정의 수업 노트에서 반복적으로 등장하는 기술/도구로, 날짜별 요약과 개념 페이지를 연결하는 기준점이다.

## 핵심 기능 / 특징

- 선택자 `$()`
- 이벤트 처리
- 속성/스타일 변경
- method chaining

# CSS

## 무엇인가

HTML 요소의 색, 크기, 배치, 글꼴 등 표현을 담당하는 스타일 언어다.

## 이 위키에서의 맥락

국비지원 웹서비스 풀스택 과정의 수업 노트에서 반복적으로 등장하는 기술/도구로, 날짜별 요약과 개념 페이지를 연결하는 기준점이다.

## 핵심 기능 / 특징

- 선택자
- 박스 모델
- position과 레이아웃
- transition과 pseudo-class

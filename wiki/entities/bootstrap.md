# Bootstrap

## 무엇인가

반응형 UI를 빠르게 만들기 위한 프론트엔드 프레임워크/라이브러리다.

## 이 위키에서의 맥락

국비지원 웹서비스 풀스택 과정의 수업 노트에서 반복적으로 등장하는 기술/도구로, 날짜별 요약과 개념 페이지를 연결하는 기준점이다.

## 핵심 기능 / 특징

- CDN 사용
- container/grid/table/card/form
- 모바일 우선 반응형 레이아웃

- 관련 수업 요약 페이지를 통해 날짜별 학습 흐름을 추적한다.

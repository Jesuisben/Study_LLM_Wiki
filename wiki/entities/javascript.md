# JavaScript

## 무엇인가

브라우저에서 웹 페이지의 동작과 상호작용을 구현하는 프로그래밍 언어다.

## 이 위키에서의 맥락

국비지원 웹서비스 풀스택 과정의 수업 노트에서 반복적으로 등장하는 기술/도구로, 날짜별 요약과 개념 페이지를 연결하는 기준점이다.

## 핵심 기능 / 특징

- 변수, 배열, 객체
- DOM 조작
- 이벤트 처리
- URL 파라미터와 페이지 동작

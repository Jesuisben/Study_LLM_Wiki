# Oracle Database

## 무엇인가

관계형 데이터베이스 관리 시스템으로, 이 과정에서는 SQL과 DB 모델링 학습의 중심 도구다.

## 이 위키에서의 맥락

국비지원 웹서비스 풀스택 과정의 수업 노트에서 반복적으로 등장하는 기술/도구로, 날짜별 요약과 개념 페이지를 연결하는 기준점이다.

## 핵심 기능 / 특징

- DBMS와 SQL 기본
- 테이블, 컬럼, 행
- 제약조건, 시퀀스, 함수, 조인, 서브쿼리
- ERD, 정규화, View, Index

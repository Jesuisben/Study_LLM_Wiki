# 2026-03-09 Java 클래스와 객체

## 한 줄 요약

사용자 정의 타입으로서 클래스, 객체, 멤버 변수, 메서드, getter/setter를 배운 날이다.

## 배운 내용

- 클래스는 기존 자료형을 조합해 새롭게 만드는 사용자 정의 타입이다.
- 객체는 클래스라는 틀로부터 만들어진 구체적인 값/대상이다.
- 멤버 변수와 메서드를 통해 객체가 데이터와 동작을 함께 갖도록 설계했다.
- `private` 필드에 접근하기 위해 getter/setter를 사용하는 캡슐화 흐름을 다뤘다.

# 2026-03-10 Java 생성자, 오버로딩, 상속 입문

## 한 줄 요약

생성자와 오버로딩을 중심으로 객체 초기화와 상속의 입문 개념을 배운 날이다.

## 배운 내용

- 생성자는 객체 생성 시 한 번 호출되어 멤버 변수 초기화에 쓰이는 특수 메서드다.
- 개발자가 생성자를 직접 만들면 눈에 보이지 않는 기본 생성자가 사라진다는 점을 확인했다.
- 오버로딩은 같은 이름의 메서드/생성자를 매개변수 개수나 타입으로 구분하는 방식이다.
- 객체 배열, 일반화, 상속, 오버라이딩으로 객체지향 설계의 다음 단계에 진입했다.

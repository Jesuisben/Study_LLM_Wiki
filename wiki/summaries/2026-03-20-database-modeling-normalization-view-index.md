# 2026-03-20 데이터 모델링, 정규화, View, Index

## 한 줄 요약

ERD, 데이터 모델링, 정규화, View, Index를 통해 DB 설계 관점을 배운 날이다.

## 배운 내용

- ERD는 엔티티, 속성, 관계를 그림으로 표현하는 데이터 모델링 도구다.
- 이상 현상을 줄이고 응집도를 높이기 위해 정규화를 학습했다.
- 기본키와 외래키, 함수 종속, 이행적 함수 종속을 구분했다.
- View와 데이터 사전, 인덱스를 통해 조회 편의성과 성능/관리 개념을 다뤘다.

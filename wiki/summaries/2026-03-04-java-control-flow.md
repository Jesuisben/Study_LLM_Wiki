# 2026-03-04 Java 제어문과 조건문

## 한 줄 요약

단순 if부터 양자택일 조건문까지 제어문의 기본 흐름을 배운 날이다.

## 배운 내용

- 프로그램은 기본적으로 위에서 아래로 실행되지만, 제어문을 쓰면 실행 흐름을 바꿀 수 있다.
- `if`는 조건이 참일 때만 특정 블록을 실행한다.
- `if-else`는 참/거짓 두 갈래 중 하나를 선택한다.
- 조건식에는 비교 연산자와 논리 연산자를 함께 사용한다.

# 2026-03-17 Oracle DDL/DML, 제약조건, 시퀀스

## 한 줄 요약

테이블 조작과 데이터 조작, 제약조건, 시퀀스를 실습한 날이다.

## 배운 내용

- DDL은 테이블 같은 데이터베이스 구조를 정의·변경하는 명령이다.
- DML은 테이블 안의 데이터를 조회·삽입·수정·삭제하는 명령이다.
- 제약조건은 잘못된 데이터가 들어오지 않게 막는 데이터베이스의 규칙이다.
- 시퀀스는 자동 증가 번호처럼 순차적인 값을 만들어 기본키 생성 등에 활용된다.

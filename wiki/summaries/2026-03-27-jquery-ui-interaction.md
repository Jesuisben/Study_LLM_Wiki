# 2026-03-27 jQuery와 UI 상호작용

## 한 줄 요약

jQuery, CSS transition, pseudo-class를 이용해 이미지/토글 UI 상호작용을 구현한 날이다.

## 배운 내용

- jQuery는 JavaScript 기반 라이브러리이며 선택자와 이벤트 처리를 짧게 작성하게 해준다.
- Bootstrap과 jQuery의 관계, CDN 로딩 방식을 확인했다.
- CSS `transition`, `transform`, pseudo-class로 hover 같은 시각 효과를 다뤘다.
- jQuery 선택자, `attr`, 토글, 카테고리 필터, 이미지 회전, method chaining을 실습했다.

# 2026-02-27 Java 기본 자료형과 연산자

## 한 줄 요약

Java의 기본 자료형, 변수 선언, 산술/대입/복합 대입/증감 연산자를 실습하고, 오후에는 GitHub 초기 설정과 원격 저장소 연결 문제를 다뤘다.

## 배운 내용

### 출력문과 계산식

첫 시간에는 변수 `a`, `b`, `c`를 두고 `2*a + 3*b - c` 같은 계산식을 출력 결과와 함께 맞춰 보는 연습을 했다.

이 실습은 단순히 계산만 하는 것이 아니라, Java에서 변수에 값을 담고 계산 결과를 사람이 읽을 수 있는 문자열과 함께 출력하는 흐름을 익히는 데 목적이 있다.

### Java 기본 자료형

수업에서는 [[concepts/java-basic-types|Java 기본 자료형]]을 처음 정리했다.

- `char`: 문자형. 한 글자만 저장한다.
- `String`: 문자열형. 0개 이상의 문자를 저장한다.
- `int`: 정수형.
- `double`: 실수형.
- `boolean`: 논리형. `true` 또는 `false` 값을 가진다.

여러 변수를 같은 타입으로 한 번에 선언하는 문법도 함께 다뤘다.

### 산술 연산자와 연산 우선순위

[[concepts/java-operators|Java 연산자]] 중 산술 연산자를 실습했다.

- `+`: 더하기
- `-`: 빼기
- `*`: 곱하기
- `/`: 나누기
- `%`: 나머지

원본에서는 Java 교안 p.17을 근거로 연산자의 우선순위를 “산술 > 대입”으로 정리했다. 계산식에서 먼저 계산해야 하는 부분은 소괄호 `()`로 묶어 우선순위를 명확히 할 수 있다.

### 대입 연산자와 복합 대입 연산자

대입 연산자는 오른쪽 값을 왼쪽 변수에 저장한다. 복합 대입 연산자는 기존 변수 값에 연산을 적용한 뒤 다시 같은 변수에 저장하는 축약 표현이다.

예를 들어 `x += 3`은 `x = x + 3`과 같은 의미다.

수업에서는 다음 연산을 실습했다.

- `+=`
- `-=`
- `*=`
- `%=`

### 증감 연산자

증감 연산자 `++`, `--`는 변수 값을 1 증가시키거나 1 감소시킨다.

- 전위 증감: `++a`, `--a`처럼 연산 전에 값이 먼저 바뀐다.
- 후위 증감: `a++`, `a--`처럼 현재 값을 먼저 사용한 뒤 값이 바뀐다.

원본에는 `++a + b++`, `a++ + --b`, `--a + --b`처럼 전위/후위가 섞인 예제가 나온다. 이런 코드는 실행 순서를 헷갈리기 쉬우므로, 학습 단계에서는 각 변수 값이 언제 바뀌는지 한 줄씩 추적하는 것이 중요하다.

### GitHub 초기 설정과 원격 저장소 문제

5~6교시에는 [[summaries/2026-02-27-github-initial-setup|GitHub 초기 설정]]을 설명했고, 7교시에는 GitHub에서 최초 소스코드를 가져오는 `Clone Repository` 흐름이 등장했다.

원본에는 로컬 저장소와 원격 저장소의 시작점이 달라 `git pull`이 거부되는 상황도 적혀 있다. 이때 `--allow-unrelated-histories`, `git push -f`, `git reset --hard` 같은 해결책이 함께 적혀 있지만, `push -f`와 `reset --hard`는 원격 또는 로컬 내용을 잃을 수 있어 실제 프로젝트에서는 매우 조심해야 한다.

## 실습 / 예제

### 평균 계산

이름, 국어 점수, 영어 점수, 평균을 출력하는 실습이 있었다. 여기서는 `double`을 써서 소수점이 있는 평균을 표현하는 흐름과 연결된다.

### 산술 연산자 출력

`x = 14`, `y = 5`일 때 더하기, 빼기, 곱하기, 나누기, 나머지를 출력하는 예제를 통해 산술 연산자의 결과 차이를 확인했다.

특히 정수끼리 나누면 `14 / 5`의 결과가 `2`가 되고, `14 % 5`의 결과가 `4`가 된다는 점이 중요하다.

### 증감 연산자 추적

전위/후위 증감 연산자가 섞인 계산에서는 다음 순서로 보면 좋다.

1. 전위 증감이 있으면 먼저 변수 값을 바꾼다.
2. 현재 표현식에서 사용할 값을 정한다.
3. 계산과 대입을 수행한다.
4. 후위 증감이 있으면 표현식에 사용한 뒤 변수 값을 바꾼다.

## 헷갈린 점 / 질문

- `char`는 한 글자, `String`은 여러 글자까지 가능한 문자열이다.
- `/`와 `%`는 둘 다 나눗셈과 관련 있지만, `/`는 몫을 구하고 `%`는 나머지를 구한다.
- `++a`와 `a++`는 단독으로 쓰면 결과가 비슷해 보이지만, 다른 연산식 안에서는 값이 바뀌는 시점이 달라 결과가 달라질 수 있다.
- `git push -f`, `git reset --hard`는 위험한 명령이므로 원본에 적혀 있더라도 실제 사용 전에는 현재 작업물과 원격 저장소 상태를 반드시 확인해야 한다.

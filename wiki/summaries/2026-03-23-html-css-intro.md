# 2026-03-23 HTML/CSS와 웹 UI 입문

## 한 줄 요약

웹 서비스의 데이터 흐름과 HTML/CSS 기본 문법을 배우기 시작한 날이다.

## 배운 내용

- 프론트엔드, 백엔드, 데이터베이스 사이의 요청/응답 흐름을 개괄했다.
- HTML은 Hyper Text Markup Language로 웹 문서의 구조를 표현한다.
- 태그, 속성, 문자 엔티티, 색상 표기법, 목록, 표, 이미지 대체 텍스트를 다뤘다.
- CSS 선택자, class/id 선택자, top-down 방식으로 스타일을 적용했다.

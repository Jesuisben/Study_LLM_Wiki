# 2026-03-05 Java for문과 while문

## 한 줄 요약

반복문 `for`, `while`을 이용해 누적 합계와 홀짝 합계를 계산한 날이다.

## 배운 내용

- `for`문은 초기식, 조건식, 증감식을 한 줄에 모아 반복 횟수를 관리한다.
- 1부터 10까지의 합, 특정 범위의 합, 홀수/짝수 합계 계산을 실습했다.
- `while`문은 조건식이 참인 동안 반복되며, 반복 횟수가 고정되지 않은 상황에 자주 쓴다.
- 세미콜론 위치와 증감식 작성이 반복문의 핵심 실수 포인트로 정리된다.

# 2026-03-03 Java 논리 연산자와 조건 연산자

## 한 줄 요약

Java 연산자 학습을 비교/논리/부정/조건 연산자로 확장한 날이다.

## 배운 내용

- 관계 연산자 `<`, `<=`, `>`, `>=`, `==`, `!=`로 값을 비교했다.
- 논리 연산자 `&&`, `||`, `!`를 조합해 여러 조건을 한 번에 판단했다.
- 계절 판별 문제처럼 범위 조건을 boolean 식으로 표현하는 연습을 했다.
- 삼항 연산자 `조건 ? 참 : 거짓`으로 간단한 선택 로직을 한 줄로 작성했다.

# HTML/CSS 기본

## 정의

HTML은 웹 문서의 구조를, CSS는 그 구조의 표현과 배치를 담당한다.

## 왜 중요한가

초보 개발 단계에서는 문법을 외우는 것보다 이 개념이 어떤 문제를 해결하는지 이해하는 것이 중요하다. 이 개념은 이후 실습, 프로젝트, 프레임워크 학습에서 반복해서 등장한다.

## 핵심 설명

- HTML은 태그와 속성으로 문서 구조를 만든다.
- CSS는 선택자와 속성으로 색, 크기, 배치, 글꼴 등을 지정한다.
- class는 여러 요소를 묶어 스타일링하고, id는 특정 요소 하나를 식별하는 데 자주 쓴다.
- 박스 모델과 position은 레이아웃 이해의 기초다.

## 예시

- 관련 수업 요약의 실습 코드와 설명을 함께 보면 좋다.

## 자주 헷갈리는 점

- 원본 수업 노트에 표시된 `중요`, `추가 공부`, 질문 표시가 있는 부분은 추후 보강 대상이다.

# Java 반복문

## 정의

같은 작업을 여러 번 실행하기 위해 사용하는 제어문이다.

## 왜 중요한가

초보 개발 단계에서는 문법을 외우는 것보다 이 개념이 어떤 문제를 해결하는지 이해하는 것이 중요하다. 이 개념은 이후 실습, 프로젝트, 프레임워크 학습에서 반복해서 등장한다.

## 핵심 설명

- `for`는 반복 횟수와 증감 규칙이 분명할 때 좋다.
- `while`은 종료 조건이 외부 입력이나 상태 변화에 따라 달라질 때 좋다.
- 무한 반복은 반드시 종료 조건이나 `break` 같은 탈출 구조를 고려해야 한다.
- 반복문에서는 초기값, 조건식, 증감식, 누적 변수의 위치가 중요하다.

## 예시

- 관련 수업 요약의 실습 코드와 설명을 함께 보면 좋다.

## 자주 헷갈리는 점

- 원본 수업 노트에 표시된 `중요`, `추가 공부`, 질문 표시가 있는 부분은 추후 보강 대상이다.

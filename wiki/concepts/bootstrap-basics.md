# Bootstrap 기본

## 정의

반응형 웹 UI를 빠르게 만들 수 있도록 미리 준비된 CSS/JS 기반 프론트엔드 프레임워크다.

## 왜 중요한가

초보 개발 단계에서는 문법을 외우는 것보다 이 개념이 어떤 문제를 해결하는지 이해하는 것이 중요하다. 이 개념은 이후 실습, 프로젝트, 프레임워크 학습에서 반복해서 등장한다.

## 핵심 설명

- CDN으로 Bootstrap 파일을 불러올 수 있다.
- container, grid, table, card, form 같은 컴포넌트를 조합해 화면을 만든다.
- 모바일 우선 반응형 설계를 지원한다.
- 직접 CSS를 모두 작성하지 않고도 일관된 UI를 빠르게 만들 수 있다.

## 예시

- 관련 수업 요약의 실습 코드와 설명을 함께 보면 좋다.

## 자주 헷갈리는 점

- 원본 수업 노트에 표시된 `중요`, `추가 공부`, 질문 표시가 있는 부분은 추후 보강 대상이다.

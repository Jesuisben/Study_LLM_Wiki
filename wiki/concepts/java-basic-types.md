# Java 기본 자료형

## 정의

Java 기본 자료형은 변수에 저장할 값의 종류를 정하는 타입이다. 수업에서는 `char`, `String`, `int`, `double`, `boolean`을 처음 구분했다.

엄밀히 말하면 `String`은 Java의 기본형(primitive type)은 아니고 문자열을 표현하는 클래스다. 다만 입문 단계에서는 “문자열을 담는 타입”으로 함께 배우는 경우가 많다.

## 왜 중요한가

컴퓨터는 값의 종류에 따라 저장 방식과 가능한 연산이 달라진다. 예를 들어 숫자는 계산할 수 있지만, 문자열은 보통 이어 붙이거나 출력하는 데 쓴다. 변수 선언 시 타입을 정하면 Java 컴파일러가 잘못된 값 사용을 미리 잡아 줄 수 있다.

## 핵심 설명

| 타입 | 의미 | 값 예시 | 수업 맥락 |
|---|---|---|---|
| `char` | 문자 하나 | `'A'`, `'가'` | 한 글자만 저장 |
| `String` | 문자열 | `"홍길동"` | 0개 이상의 문자를 저장 |
| `int` | 정수 | `3`, `40` | 나이, 횟수, 정수 계산 |
| `double` | 실수 | `41.5` | 평균처럼 소수점이 있는 값 |
| `boolean` | 논리값 | `true`, `false` | 참/거짓 판단 |

같은 타입의 변수는 한 줄에 여러 개 선언할 수 있다.

예: `int age, time;`

## 예시

수업에서는 이름, 국어 점수, 영어 점수, 평균을 출력하는 예제를 통해 문자열과 숫자 타입을 함께 사용했다.

- 이름: 문자열이므로 `String`과 연결된다.
- 국어/영어 점수: 정수 또는 실수로 표현할 수 있다.
- 평균: 소수점이 나올 수 있으므로 `double`과 연결된다.

## 자주 헷갈리는 점

- `char`는 작은따옴표를 쓰고 한 글자만 담는다.
- `String`은 큰따옴표를 쓰고 여러 글자까지 담을 수 있다.
- `int`끼리 나누면 소수점이 버려질 수 있다. 평균처럼 소수점이 필요한 계산은 `double`을 고려해야 한다.
- `String`은 입문 수업에서는 자료형처럼 배우지만, 실제 Java 분류에서는 기본형이 아니라 참조형이다.

# Java 연산자

## 정의

Java 연산자는 값이나 변수를 계산, 비교, 대입할 때 쓰는 기호다. 2026-02-27 수업에서는 산술 연산자, 대입 연산자, 복합 대입 연산자, 증감 연산자를 중심으로 다뤘다.

## 왜 중요한가

프로그램은 값을 저장하고, 계산하고, 결과를 다시 저장하는 과정을 반복한다. 연산자를 이해하면 변수 값이 어떻게 바뀌는지 추적할 수 있고, 출력 결과를 예측할 수 있다.

## 핵심 설명

### 산술 연산자

| 연산자 | 의미 | 예시 |
|---|---|---|
| `+` | 더하기 | `x + y` |
| `-` | 빼기 | `x - y` |
| `*` | 곱하기 | `x * y` |
| `/` | 나누기 | `x / y` |
| `%` | 나머지 | `x % y` |

정수 `14`와 `5`를 나누면 `14 / 5`는 몫 `2`, `14 % 5`는 나머지 `4`가 된다.

### 대입 연산자

`=`는 오른쪽 값을 왼쪽 변수에 저장한다. 수학의 “같다”와 다르게, 프로그래밍에서는 “대입한다”는 의미로 이해해야 한다.

### 복합 대입 연산자

복합 대입 연산자는 기존 변수 값에 연산을 적용한 뒤 다시 저장하는 축약 표현이다.

- `x += 3`은 `x = x + 3`
- `x *= 4`는 `x = x * 4`
- `x %= 5`는 `x = x % 5`

### 증감 연산자

`++`는 1 증가, `--`는 1 감소를 뜻한다.

- 전위: `++a`, `--a` — 값을 먼저 바꾸고 식에 사용한다.
- 후위: `a++`, `a--` — 식에 먼저 사용하고 나중에 값을 바꾼다.

## 예시

`c = ++a + b++` 같은 식에서는 다음처럼 추적한다.

1. `++a` 때문에 `a`가 먼저 1 증가한다.
2. 증가한 `a` 값이 계산에 사용된다.
3. `b++`는 현재 `b` 값이 계산에 먼저 사용된다.
4. 계산이 끝난 뒤 `b`가 1 증가한다.
5. 계산 결과가 `c`에 대입된다.

## 자주 헷갈리는 점

- `=`는 비교가 아니라 대입이다.
- `/`는 나눗셈의 몫, `%`는 나머지를 구한다.
- 전위 증감과 후위 증감은 단독으로 쓰면 비슷해 보이지만, 다른 연산식 안에서는 결과가 달라질 수 있다.
- 복잡한 증감 연산식은 실무 코드에서는 가독성을 해칠 수 있으므로, 학습할 때는 순서 추적용으로 이해하는 것이 좋다.

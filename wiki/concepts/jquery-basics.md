# jQuery 기본

## 정의

JavaScript DOM 선택과 이벤트 처리를 짧고 편하게 작성하도록 돕는 라이브러리다.

## 왜 중요한가

초보 개발 단계에서는 문법을 외우는 것보다 이 개념이 어떤 문제를 해결하는지 이해하는 것이 중요하다. 이 개념은 이후 실습, 프로젝트, 프레임워크 학습에서 반복해서 등장한다.

## 핵심 설명

- `$()` 선택자로 HTML 요소를 찾는다.
- 이벤트, 속성 변경, 스타일 변경, 애니메이션성 효과를 간결하게 작성할 수 있다.
- `attr`은 HTML 속성을 읽거나 바꿀 때 사용한다.
- method chaining으로 여러 동작을 이어서 표현할 수 있다.

## 예시

- 관련 수업 요약의 실습 코드와 설명을 함께 보면 좋다.

## 자주 헷갈리는 점

- 원본 수업 노트에 표시된 `중요`, `추가 공부`, 질문 표시가 있는 부분은 추후 보강 대상이다.

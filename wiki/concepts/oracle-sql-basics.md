# Oracle SQL 기본

## 정의

Oracle Database에 데이터를 조회·정의·조작하도록 요청하는 SQL의 기본 구조다.

## 왜 중요한가

초보 개발 단계에서는 문법을 외우는 것보다 이 개념이 어떤 문제를 해결하는지 이해하는 것이 중요하다. 이 개념은 이후 실습, 프로젝트, 프레임워크 학습에서 반복해서 등장한다.

## 핵심 설명

- 테이블은 행(row)과 열(column)로 데이터를 저장한다.
- `SELECT`는 데이터를 조회한다.
- DDL은 구조를 만들고 바꾸는 명령, DML은 데이터를 다루는 명령이다.
- DBeaver 같은 도구를 통해 SQL을 실행하고 결과를 확인한다.

## 예시

- 관련 수업 요약의 실습 코드와 설명을 함께 보면 좋다.

## 자주 헷갈리는 점

- 원본 수업 노트에 표시된 `중요`, `추가 공부`, 질문 표시가 있는 부분은 추후 보강 대상이다.
